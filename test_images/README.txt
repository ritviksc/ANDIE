Here are some files that you can use for testing your ANDIE project.

- clocktower.png is the same file that is used in the examples in the lab book.

- test-image.png is the primary image that Mark George uses when marking your project.  Several groups have asked for a copy of this image, so here you go.

  The box with the 'A' in it is fully transparent.  The gradient under the word 'TESTING' fades from black to fully transparent.  The box in the bottom left contains some random noise.

  Note: You should use more than just this image when testing your project.

- graph-paper.pdf is not a test image.  It is an A4 page with a 5mm grid on it that you can print and scribble on to help you to understand and plan what needs to happen to the pixels in the image when implementing the various image operations.
